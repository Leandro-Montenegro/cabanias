🏡 Sistema de Gestión para Complejo de Cabañas

Este proyecto es una aplicación web para la administración de un complejo de cabañas.
El sistema permite al propietario gestionar información relacionada con su negocio a través de una interfaz web.

La aplicación está compuesta por:

Frontend desarrollado en TypeScript

Backend desarrollado en Java

Esta arquitectura permite separar la interfaz de usuario de la lógica del servidor.

🚀 Tecnologías utilizadas
Frontend

TypeScript

HTML

CSS

Backend

Java

Herramientas

Git

GitHub

📂 Estructura del proyecto
/cabanias
│
├── backend
│   └── (código del servidor en Java)
│
├── frontend
│   └── (interfaz desarrollada en TypeScript)
│
└── README.md

Descripción

backend/ → lógica del servidor y procesamiento de datos

frontend/ → interfaz web con la que interactúa el usuario

⚙️ Uso

Clonar el repositorio

git clone https://github.com/Leandro-Montenegro/cabanias

Entrar a la carpeta del proyecto

cd cabanias

Ejecutar el backend en Java y abrir el frontend en el navegador.

📋 Funcionalidad

El sistema está pensado para ayudar al propietario del complejo a:

Administrar información del complejo de cabañas

Gestionar datos del negocio desde una interfaz web

Organizar la información mediante un sistema cliente-servidor
